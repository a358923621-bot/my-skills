# Meituan Review Response Templates

## Warm Sister Style (Positive Reviews 4-5 stars)

### For Food Quality Praise

```
亲爱的，太开心看到你喜欢我们家的[菜品名]啦！谢谢你的认可😊~

我们家的[特色]都是坚持现炒现做的，这样才能保持最佳的口感和味道。能得到你的喜欢真的很开心！

期待你的下次光临，记得跟店员说多加点[小菜/米饭]，姐姐偷偷给你多加一勺😉
```

### For Service Praise

```
哇！看到你对服务的评价这么高，姐姐心里暖暖的😊~

我们一直相信，用真心对待每一位客人，客人一定能感受到。你的认可就是对我们要最大的鼓励！

常来坐坐，姐姐给你留最好的位置～
```

### For Atmosphere Praise

```
看到你喜欢我们店的环境，姐姐好开心！😊

我们装修的时候就想着，要让客人有回家的感觉，舒服又温暖。你能感受到，我们的努力就没白费啦！

期待常来，在这里就是给自己放松的时间～
```

### General Positive Review

```
亲爱的，看到你这么用心的评价，姐姐真的很感动！😊~

谢谢你的喜欢和认可，这些都是我们应该做的。能得到你的满意就是我们最大的动力！

期待下次见到你，给你准备小惊喜～😉
```

## Sincere Problem-Solving Style (Negative/Mixed Reviews 0.5-3.5 stars)

### For Food Quality Issues

```
非常扎心，但也非常感谢您的直言不讳。

看到您对菜品质量的反馈，我们心里很难受。每一道菜都应该以最好的状态呈现给客人，这次明显没有达到标准。

我们非常重视您的反馈，会立即后厨检查问题。实在不想因为这一次的糟糕体验，就失去您这位朋友。

希望能有个机会，邀请您再次到店免费体验一次，让我们把坏事变好事，用真正的出品重新赢回您的认可！
```

### For Service Issues

```
看到这条评价，我们心里非常难受。

服务不到位是我们的责任，没有任何借口。让本该愉快的用餐体验变得不愉快，我们深感抱歉。

我们会对服务团队进行重新培训，确保不再出现类似问题。如果您愿意再给我们一次机会，我们会用最真诚的服务让您感受到改变。

期待您的再次光临！
```

### For Price Concerns

```
非常感谢您的坦诚反馈。

关于价格，确实如您所说，我们的定价不算低。这是因为我们坚持使用新鲜优质的食材，成本确实较高。但这不能成为让客人觉得不值的原因。

您的建议我们会认真考虑，看看如何在保证品质的前提下，给客人更实在的价值。

希望您能理解，也期待您再次光临时，能感受到我们的用心！
```

### For Waiting Time Issues

```
非常抱歉让您等了这么久！

无论什么原因，让客人等待太久都是我们的失职。我们理解等待时的心情，换位思考，我们自己也会不开心。

我们会优化排队和出餐流程，尽量缩短等待时间。真的非常抱歉！

如果您愿意再给我们一次机会，我们会优先安排您的订单，减少您的等待。期待您的原谅！
```
